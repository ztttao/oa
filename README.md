# oa
* 主要运用Mybatis、Spring、SpringMVC和Mysql。<br>
* OA是公司内部员工使用的办公系统，为方便管理办公系统主要功能模块为部门信息管理、员工信息管理、和报销单处理。界面简洁，处理方便。方便查询部门信息和员工信息，以及出差等产生的需报销的报销单处理<br>

<br>


目前已有的功能为：
---------------
* 员工的登陆注销<br>
* 员工和部门的添加和修改<br>
* 报销单的提交,修改<br>
* 报销单的审核和退回<br>
